Time policy

True solar time is useful only when the birth context actually calls for it.

In Korean Saju, the hour pillar can be sensitive. But blindly converting every birth to true solar time can create a false sense of precision. A cleaner reading starts with local civil time and checks whether longitude, timezone, and DST justify an adjustment.

How the report handles time

Default: local civil time

The recorded time at the birthplace is the first anchor. This avoids changing the chart before the birth record and local clock context are understood.

Check timezone rules

The city and country are reviewed so a Korea-only time assumption does not slip into the chart.

Check daylight saving context

DST can affect the recorded hour for some regions and birth years.

Check birthplace position when relevant

If true solar time could move the hour boundary, the report notes the practical effect. Buyers do not need to look up decimal coordinates before ordering.

When this matters most

This matters most for overseas births, cities near timezone edges, and birth times close to a two-hour branch boundary. The Premium report includes this check so the interpretation is not built on the wrong clock assumption.
